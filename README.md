# Javascript Notes

This app was developed as a final course project
"Full Stack JavaScript Programmer in 8 weeks"

**Features:** 
  - Register account;
  - Login;
  - Home with personal notes;
  - Logout;
  - Edit email, name and password;
  - Delete account.
